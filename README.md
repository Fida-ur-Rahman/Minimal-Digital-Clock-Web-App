#  Modern Digital Clock

A responsive **Digital Clock Web Application** built using **HTML, CSS, and JavaScript** that displays the current time in real-time. The interface is clean, minimal, and updates every second to ensure accurate time display.

---

##  Features

*  Real-time clock updates every second
*  Modern and minimal user interface
*  Fully responsive design (mobile, tablet, desktop)
*  Easy to customize styles and colors
*  Lightweight and fast performance

##  How It Works

The clock uses JavaScript's built-in **Date object** to fetch the current system time.
A function runs every second using `setInterval()` to update the hours, minutes, and seconds displayed on the screen.

##  Customization

You can easily customize the clock by:

* Changing fonts and colors in **style.css**
* Modifying time format (12-hour / 24-hour) in **script.js**
* Adding date or day display
